インストール

インストーラ版(Windows)

インストール方法

smoothcsv〜setup.exeを実行します。

アンインストール方法

Windows XP:「スタート」＞「コントロールパネル」＞「プログラムの追加と削除」を開き、「SmoothCSV」を選択して「削除」します。
Windows Vista、7:「スタート」＞「コントロールパネル」＞「プログラムのアンインストール」（または「プログラムと機能」）を開き、「SmoothCSV」をダブルクリックします。

Zip版(Windows)

インストール方法

zipファイルを任意のフォルダに解凍します。 バージョンアップするときは、旧バージョンのフォルダに上書きで 解凍します。

アンインストール方法

フォルダごと削除します。
